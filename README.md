# How To Run This Project

- ### Run Composer Install

```bash
composer install
```

- ### Edit .env file

- ### Run Migration

```bash
php artisan migrate:fresh --seed 
```

- ### Run Storage Link

```bash
php artisan storage:link
```

- ### Run Server

```bash
php artisan serve
```
